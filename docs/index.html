Intro

The player use a cannon to destroy the pirates ships sailing. When all the ships are sinked the player wins and pass to the next level. If the cannon runs out of bullets the player lose the game.

Gameplay demo video:

Game keys

left/right arrows move cannon left-right
up/down arrows rotate the shot angle up/down
s shoot, more long key stay pressed more will be the shot strength
h toggle help
q quit game
enter start game
1,2,3,4,9 change camera view: normal, cannon, top, front, free
k/l save/load game

Features

Configurable levels

In bin/data/config/ folder are present a config.xml file to configure the game levels:

Level bullet number and ships are configurable.

Save/load game

Pressing k/l kay the app write/read and load a xml file with the game status.

Developments

  • Compute ship mesh bounding boxes dinamically
  • Enemy ships shoot on the player
  • Textures and materials for better sand and grass
  • Fbo draw for better performance
  • Use of inheritance for reducing animation loops
